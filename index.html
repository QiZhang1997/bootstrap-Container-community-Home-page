About containers

As a lightweight virtualization technology, containers are attracting much attention and widely deployed in the cloud data centers. Compared to virtual machines(VMs), containers have the following three advantages:

  • Faster. Containers can be deployed in milliseconds
  • Migrate easily. Containers can be easily migrated to other servers with compatible operating system kernels
  • Higher resource efficiency. A single operating system can host more containers than virtual machines
